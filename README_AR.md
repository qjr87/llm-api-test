## 📖 نظرة عامة

أداة اختبار API للنماذج اللغوية الكبيرة هي أداة قوية تعتمد على الويب مصممة لقياس ومقارنة أداء واجهات برمجة التطبيقات المختلفة للنماذج اللغوية الكبيرة. سواء كنت تقيم مقدمي خدمات مختلفين، أو تحسن تطبيقات الذكاء الاصطناعي الخاصة بك، أو تجري بحثاً، فإن هذه الأداة توفر مقاييس ورؤى شاملة.

## ✨ الميزات الرئيسية

### 🔌 دعم واجهات برمجة التطبيقات
- **OpenAI**: GPT-3.5، GPT-4 والنماذج الأحدث
- **Google Gemini**: Gemini Pro، Gemini Pro Vision
- **واجهات برمجة التطبيقات المخصصة**: أي نقطة نهاية متوافقة مع OpenAI

### 📊 مقاييس الأداء
- **وقت الاستجابة**: قياس زمن استجابة الرمز الأول
- **سرعة الإخراج**: حساب الرموز في الثانية
- **معدل النجاح**: تتبع موثوقية واجهة برمجة التطبيقات
- **تقييم الجودة**: أدوات مقارنة الاستجابات

### 🌐 تجربة المستخدم
- **واجهة متعددة اللغات**: دعم 7 لغات
- **تصميم متجاوب**: يعمل على سطح المكتب والهاتف المحمول
- **نتائج فورية**: مراقبة الأداء المباشر
- **تتبع التاريخ**: سجلات اختبار دائمة

### ☁️ خيارات النشر
- **التطوير المحلي**: إعداد خادم HTTP بسيط
- **الاستضافة الثابتة**: متوافق مع أي مضيف ثابت

## 🚀 البدء السريع

### المتطلبات الأساسية
- متصفح ويب حديث (Chrome، Firefox، Safari، Edge)
- Node.js و npm مثبتان
- مفاتيح API للخدمات التي تريد اختبارها

### التطوير المحلي

1. **استنساخ المستودع**
   ```bash
   git clone https://github.com/qjr87/llm-api-test.git
   cd llm-api-test
   ```

2. **تثبيت التبعيات وبدء الخادم**
   ```bash
   npm install
   npm start
   ```

   **الطرق البديلة:**
   ```bash
   # باستخدام Python 3
   python -m http.server 8000
   
   # باستخدام PHP
   php -S localhost:8000
   ```

3. **فتح في المتصفح**
   انتقل إلى `http://localhost:8000`

### 🔧 دليل التكوين

#### إعداد واجهة برمجة التطبيقات
1. **اختيار البروتوكول**: اختر مقدم واجهة برمجة التطبيقات (OpenAI، Gemini، أو مخصص)
2. **إدخال نقطة النهاية**: رابط واجهة برمجة التطبيقات (يتم ملؤه تلقائياً للمقدمين القياسيين)
3. **إضافة مفتاح API**: مفتاح المصادقة الخاص بك
4. **تكوين النماذج**: حدد النماذج المراد اختبارها

#### معاملات الاختبار
- **جولات الاختبار**: عدد التكرارات لكل نموذج
- **المطالبات**: مطالبات اختبار مخصصة أو استخدام الافتراضية
- **التزامن**: معالجة الطلبات المتوازية

#### مثال على التكوين
```javascript
// تكوين OpenAI
البروتوكول: "openai"
رابط API: "https://api.openai.com/v1/chat/completions"
مفتاح API: "sk-..."
النماذج: "gpt-3.5-turbo,gpt-4"

// تكوين Gemini
البروتوكول: "gemini"
رابط API: "https://generativelanguage.googleapis.com/v1beta/models"
النماذج: "gemini-pro"
```

## 🚀 النشر

### الاستضافة الثابتة

انشر على أي خدمة استضافة ثابتة:

- **Vercel**: `vercel --prod`
- **Netlify**: اسحب وأفلت مجلد المشروع
- **GitHub Pages**: فعّل في إعدادات المستودع
- **Firebase Hosting**: `firebase deploy`

### نشر Docker

```dockerfile
FROM nginx:alpine
COPY . /usr/share/nginx/html
EXPOSE 80
CMD ["nginx", "-g", "daemon off;"]
```

```bash
docker build -t llm-api-test .
docker run -p 8080:80 llm-api-test
```
